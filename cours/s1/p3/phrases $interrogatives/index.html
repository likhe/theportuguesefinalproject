La forme interrogative

Présentation

Invariable

quemPour indiquer une personne.

quePour indiquer une personne ou un objet.

quandoPour demander le moment.

ondePour demander le lieu.

comoPour indiquer un mode, une manière.

porquePour indiquer la raison, le motif, la cause.

Variable

quantoPour indiquer les quantités et les portions.

qualPour indiquer précisément le sujet discuté.
