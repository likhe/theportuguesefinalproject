Le pluriel

Singulier Pluriel Les exemples
terminant par une voyelle -s aluno -> alunos, gata->gatas
-m -ns homem -> homens, bombom -> bombons
-ao, -a -aos, -as fa -> fas, alemao -> alemaos
-r,-z,-s -es dor -> dores, vez -> vezes, burguês -> burgueses
-al, -el, -ol -is normal -> normais, cruel -> crueis, farol -> farois
-il (oxitone) -is juvenil -> juvenis
-il (paroxitone) -eis fragil -> frageis
