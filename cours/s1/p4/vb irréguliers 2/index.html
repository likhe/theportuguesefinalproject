Le futur proche

Présentation

Pour exprimer une action qui va se dérouler dans un avenir très proche du présent, on emploie le futur proche. Pour faire un futur proche, il suffit d'ajouter avant le verbe à l'infinitif le verbe ir.

Ex: Eu vou conduzir. (Je vais conduire)
