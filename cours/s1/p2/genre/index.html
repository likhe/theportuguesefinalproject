Le féminin et le masculin en portugais

Masculin Féminin Les exemples
-o -a aluno -> aluna, gato->gata
se termine par une consonne -a autor -> autora, burguês -> burguesa
-ao -oa,-a, -ona leao -> leoa, alemao -> alema, comilao -> comilona
-or -eira bordor -> bordadeira
-dor, -tor -triz imperador -> imperatriz , ator -> atriz
-eu -eia europeio -> europeia
